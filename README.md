Descrição do projeto:
O projeto Biblioteca Inteligente tem como objetivo aplicar técnicas de Inteligência Artificial (IA) para otimizar o gerenciamento de acervos e o processo de recomendação de livros em uma biblioteca universitária.
A proposta consiste em demonstrar como algoritmos de IA podem contribuir para: Automatizar processos de organização e busca; Listar documentos, adicionar e excluir documentos do acervo, etc.
Esse trabalho foi desenvolvido como parte da disciplina Programação para ciência de dados, explorando o uso prático de bibliotecas do Python.

Ferramentas e Tecnologias Utilizadas:
Python – linguagem principal
Os - listagem de arquivos
Pandas – manipulação e análise de dados
FPDF - gerar relatórios ou certificados em PDF
Google Colab – ambiente de execução em nuvem
GitHub – controle de versão e compartilhamento do projeto

Conclusão:
O estudo reforça a importância da Inteligência Artificial como instrumento de modernização e eficiência em ambientes educacionais e culturais.
A Biblioteca Inteligente representa um exemplo prático de como algoritmos simples podem gerar impacto real em processos de gestão, organização e acesso à informação.
